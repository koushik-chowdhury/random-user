# Random Users UI (React)

## About

This project is a React-based user interface that fetches random user data from a public API and displays it in a structured layout. The application presents user information such as profile image, name, email, and location using reusable components and a clean grid-based design.

---

## Tech Stack

- React (UI library)
- Vite (build tool)
- Axios (API requests)
- JavaScript (ES6+)
- CSS (custom styling)

## Api used
- https://api.freeapi.app/api/v1/public/randomusers
